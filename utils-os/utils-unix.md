# Unix

Ubuntu & co

## Directories

```
usr  -> unix system resources
bin  -> binaries
sbin -> super binaries
var  -> variable
dev  -> device
opt  -> optional
```

## Debug

```
/var/log/apache2/error.log
/var/log/rabbitmq/rabbit.log
```

## Update

```
sudo apt update
sudo apt-get update  
```
